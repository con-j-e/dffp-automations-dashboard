# About

A simple GitHub Pages dashboard displaying exit status history for scheduled tasks as well as their expected next execution times. The dashboard is primarily a tool for internal team use.

This repository is a robocopy mirror of the `gh_pages` subdirectory of the `dffp_automations_dashboard` project in [akdof-monorepo](https://github.com/con-j-e/akdof-monorepo-draft-0).
